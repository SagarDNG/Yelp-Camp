# Deployment commands for Heroku
Ensure you have Git & Heroku CLI
First, go to folder location 
```
cd /d folderPath
```
```
git init
```
```
git add .
```
```
git commit -m "commitMessageHere"
```
```
heroku login
```
```
heroku create
```
```
heroku git:remote -a herokuAppName
```
```
git push heroku master
```
